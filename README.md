# CMS. Proudly using Koa 2, Vue 2 , Webpack 4 and Babel 7

## Dependency

Mysql: [official site link](https://www.mysql.com/)

Node: ~7.6.0

## Starting

```
$ git clone https://github.com/liuyanzhi08/vue-cms.git
$ cd vue-cms
$ npm install
$ npm run db:migrate
$ npm run db:seed
$ npm start
```
